Video Context Safari extension

by Marc Hoyois

My Safari extensions > Video Context

Download Video Context 1.2

This extension adds two contextual menu items to HTML5 media elements that allow you to play the media in QuickTime Player or download it for later viewing.

You can browse the source on Github.

New in version 1.2

Compatibility with ClickToPlugin/ClickToFlash 2.2 and built-in French localization.

New in version 1.1

Supports videos created by the YouTube5 extension.

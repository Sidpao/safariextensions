This simple proof-of-concept extension adds a reload button in Safari’s toolbar that transforms into a stop button when a page is loading. It thus provides exactly the same functionality as the built-in Stop/Reload control in the address field, but in the form of a button that can be placed anywhere in the toolbar (as it used to be before Safari 4).

Known issues

The button does not work on pages restored from Safari’s cache. This is the consequence of a bug in Safari that prevents such pages from receiving messages sent by an extension’s global page; it has been reported. For this extension the bug could be circumvented, but at the expense of not remembering the scrolling state when reloading.
